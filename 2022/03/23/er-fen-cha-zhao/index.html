public int firstEqual(int arr[], int target) {
    int l = 0, r = arr.length-1;
    while(l < r) {
        int mid = l + r >> 1;
        if(check(mid)) { // arr[mid] >= target
            r = mid;
        } else {
            l = mid + 1;
        }
    }
    return l;
}

public int lastEqual(int arr[], int target) {
    int l = 0, r = arr.length - 1;
    while(l < r) {
        int mid = l + r + 1 >> 1;
        if(check(mid)) // arr[mid] <= target
            l = mid;
        else
            r = mid-1;
    }
    return l;
}

Ex. 34.在排序数组中查找元素的第一个和最后一个位置

二分是从中间开始找起的，所以找的必然是条件区间中靠近中心的的边界值。

二分本质

「二分」的本质是两段性，并非单调性。只要一段满足某个性质，另外一段不满足某个性质，就可以用「二分」

「二分」不是单纯指从有序数组中快速找某个数，这只是「二分」的一个应用。

Ex. 33. 搜索旋转排序数组

public int search(int[] nums, int target) {
    int n = nums.length, l = 0, r = n-1;
    if(n == 0) return -1;
    if(n == 1) return nums[0] == target ? 0 : -1;
    // [4, 5, 6, 7] [0, 1, 2] 分割点
    while(l < r) {
        int mid = l + r + 1 >> 1;
        if(nums[mid] >= nums[0])
            l = mid;
        else
            r = mid-1;
    }
    if(target >= nums[0])
        l = 0;
    else {
        l = l + 1;
        r = n-1;
    }
    while(l < r) {
        int mid = l + r >> 1;
        if(nums[mid] >= target) 
            r = mid;
        else
            l = mid+1;
    }

    return nums[r] == target ? r : -1;
}
