视频切片并加密

2021-06-29- 2022-03-23
150- 1m-
其他-ffmpeghls

使用ffmpeg视频切片与加密

openssl rand  16 > encrypt.key # key
openssl rand -hex 16 # IV

# 新建hls_key_info_file 
# 格式
Key URI
Path to key file  
IV

# eg enc.keyinfo
# http://localhost:8080/encrypt.key 
# encrypt.key
# IV

要保证key url可以访问到, 可以在测试目录下使用 

python -m http.server 8080 # eg http://localhost:8080/encrypt.key

生成m3u8文件

 ffmpeg -y
-i test.mp4 
-hls_time 6 # 将test.mp4分割成每个小段多少秒
-hls_key_info_file enc.keyinfo
-hls_playlist_type vod   # vod 是点播
-hls_segment_filename "test%d.ts" #  每个小段的文件名
test.m3u8   #  生成的m3u8文件

ffmpeg -y -i test.mp4 -hls_time 6 -hls_key_info_file enc.keyinfo -hls_playlist_type vod -hls_segment_filename "test%d.ts" test.m3u8
