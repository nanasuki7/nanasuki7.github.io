浮点数二分算法模板

y总模板

double bs(double l, double r) {
    // l-r>1e-(k+2),k表示的是精确到第几位
    double eps = 1e-(k+2);
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) r = mid;
        else l = mid;
    }
    return l;
}

680. 剪绳子 - AcWing题库

有 N 根绳子，第 i 根绳子长度为 Li，现在需要 M 根等长的绳子，你可以对 N 根绳子进行任意裁剪（不能拼接），请你帮忙计算出这 M 根绳子最长的长度是多少。

输入格式

第一行包含 2 个正整数 N、M，表示原始绳子的数量和需求绳子的数量。

第二行包含 N 个整数，其中第 i 个整数 Li 表示第 i 根绳子的长度。

输出格式

输出一个数字，表示裁剪后最长的长度，保留两位小数。

数据范围

1≤N,M≤100000,
0<Li<10e9

输入样例：

3 4
3 5 4

输出样例：

2.50

代码

import java.util.*;
public class Main {
    public static void main(String[] args) {
        Scanner in  = new Scanner(System.in);
        int n = in.nextInt(), m = in.nextInt();
        int arr[] = new int[n];
        for (int i = 0; i < n; i++)
            arr[i] = in.nextInt();
        double l = 0, r = 1e9;
        while(r - l > 1e-4) {
            double mid = (r+l)/2;
            if(check(arr, m, mid))
                l = mid;
            else
                r = mid;
        }
        System.out.println(String.format("%.2f", r));
        in.close();
    }

    public static boolean check(int arr[], int m, double mid) {
        int cnt = 0;
        for(int len : arr) {
            cnt += len / mid;
            if(cnt >= m)
                return true;
        }
        return false;
    }
}

要明白check的意义， cnt >= m 说明划分的太小了，所以丢掉左边
